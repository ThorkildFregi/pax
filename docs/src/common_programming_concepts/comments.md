# Comments

Do you want people to understand your code ? No, problem, you can add comments to your code and give indications to your fellow developers.

Here is a simple comment

```pax
// Hello World!
```

In Pax, a comment begins with two slashes ```//``` and end when the line breaks. So if you need comments on multiple line, you'll need to type ```//``` on each line.

```pax
// Lorem ipsum dolor sit amet, consectetur adipiscing elit.
// Praesent tincidunt tortor a lectus pellentesque, sed consequat dolor bibendum.
//Nullam porttitor. 
```

Comments can also be placed at end of line containing codes:

```pax
print("This is a print"); // This is a comment
```
