# Introduction

Welcome! You are reading the documentation for Pax. Pax is a programming language developed in Rust to ensure a fast and memory-safe environment for data science and statistics. Imagine the syntactic ease of Python (yes, elif is back!), the conciseness of C++ (How much logic you cram into a single line is between you and your God. We don't judge; we just execute.), and the type security of Rust. Add high-speed performance and advanced mathematical operations, and you have exactly what this language strives to be.

## Who Pax Is For

For now, mostly nobody. But in the future, it aims to assist mathematicians, engineers, and statisticians who need to perform long and complicated calculations.

### Students
